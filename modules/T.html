each (t, callback)
Invoke a callback over all elements of a table.

Parameters:

  • t table the table to iterate
  • callback function callback to invoke. Takes (value, key) as params

Returns:

    table t (for chaining)
map (t, mapper)
Map the elements of a table to different values (and keys).

Parameters:

  • t table the table to map
  • mapper function callback to invoke. Takes (value, key) as params and is expected to return at least one value (the mapped value) and optionally a new key.

Returns:

    table the mapped table
filter (t, filterer)
Filter the elements of a table.

Parameters:

  • t table the table to filter
  • filterer function callback to invoke. Takes (value, key) as params and it expected to return either true (if the key, value pair should be kept) or false (if the key, value pair should be discarded).

Returns:

    table the filtered table
reduce (t, reducer, reduction)
Reduce the elements of a table to a single value.

Parameters:

  • t table the table to reduce
  • reducer func callback to invoke. Takes (reduction, value, key) as params and is expected to return the reduction.
  • reduction the reduction's initial value

Returns:

    the reduction
slice (t[, istart=0[, iend=#t]])
Create a table from a consecutive subset of another table's array elements.

Parameters:

  • t table the table from which to extract
  • istart number starting index of the slice, can be negative (default 0)
  • iend number ending index of the slice, can be negative (default #t)

Returns:

    table the table slice
join (t[, sep=""])
Join the array elements of a table into a string.

Parameters:

  • t table the table to join
  • sep string optional separator (default "")

Returns:

    string the joined string
find (t, search)
Find a (value, key) pair in a table.

Parameters:

  • t table the table to search
  • search if this is a function, the first pair w/ `search(value, key) == true` is returned. Otherwise the first pair w/ `value === search` is returned. Default returns (nil, nil).

Returns:

  1. value the found value
  2. key the found key
